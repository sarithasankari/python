    6. Object-Oriented Programming (OOP)
    Classes and Objects:

    python
    class Person:
    def __init__(self, name, age):
    self.name = name
    self.age = age
    def greet(self):
    return f"Hi, I'm {self.name}."
    person = Person("Alice", 25)
    print(person.greet())
